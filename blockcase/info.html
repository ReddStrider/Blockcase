Blockcase is a one-command-generator I made that packs many minecraft command blocks into one

I made it to be able to play on servers just a bit easier ^-^


At least I haven't found another one-command-generator for 1.19.3+
Guess datapacks really killed the stuff eh?
I like working in restrictions though so I use them.

BASE USE:

Paste your commands into the grey text box
Press the command block icon to change the type of start and trail output

The "New Script" and "Erase Script" buttons add and erase new command chains respecivly

Once your command machine is made just press "Copy Command to get it to the clipboard"
You can now paste that into a command block and power it, then power the rail on top.

If you want to share it, just enter a name for your creation and press "Update Link", then just send the link!

ADVANCED USE

If your command is too complex it might not fit in a URL.
Using "Copy JSON" and "Load JSON" we can remedy that.
It wont copy name or clickEvent however

Wait clickEvent? What's that?
Basically when you click on the command sign name in front of the loaded machine it will run a command!
One might use this to give the player the necessary items for a one-command creation.
